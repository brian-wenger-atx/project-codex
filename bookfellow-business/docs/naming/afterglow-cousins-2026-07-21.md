# Afterglow cousins — reading + continuing memory (2026-07-21)

Brian likes **Afterglow**; wants names that feel more **reading-friendly** and imply **memory that continues** between sessions.

Interactive: Cursor canvas `afterglow-cousins.canvas.tsx`.

## Design target

| Afterglow has | We want more of |
|---------------|-----------------|
| Residual warmth after a sitting | Book/page/chapter language |
| Soft, human, not AI | Continuity — same book still alive in your head |

## Agent lean

- **Keep Afterglow feel, more bookish:** Afterword, Afterpage, Afterlight, Pageglow  
- **Strongest “memory continues”:** Throughline, Linger, Afterimage, Reprise, Imprint, Filament  

`.read` still Amazon-closed — use `.io` / `.cc` / `.com` (confirm availability when shortlisting).
