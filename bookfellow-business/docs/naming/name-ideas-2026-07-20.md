# Brand name ideas (100) — 2026-07-20

Working list for the commercial companion product. **Not locked.** Homelab **Codex** is taken.

**Brief:** single word preferred; not AI-wrapper sounding; companion for books you already own; recall + return-to-book (not a substitute). Suggested URLs lean `.read` / `.cc` / `.io` — **availability unchecked**.
